{{ page.title }}

LibreOffice can be used to create or edit the same, and even more, types of files as Microsoft Office all from one package.

Context

Use when a free, open source, portable, and cross-platform office suite is needed. LibreOffice should also be used when a drawing platform is needed, creating HybridPDFs, importing SVG vector or Adobe Photoshop graphics, and importing from CorelDraw, Corel Presentation Exchange, or Adobe/Macromedia Freehand.

Required

  • 1 computer with LibreOffice

Steps

  1. Open LibreOffice either from the Start menu or the desktop icon.
  2. Using the menu bar on the left, select what form of document to create.

    NOTE: Open File will need to be selected to edit an existing file.

Post-requisites

  • Uninstall Microsoft Office from your computer.
